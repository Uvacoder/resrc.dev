Resrc is built with [Gastby](https://gatsbyjs.org/) and [Chakra UI](https://chakra-ui.com/).

1. Install dependencies: `yarn install`
2. Copy env: `cp example.env .env` (set API keys in `.env`)
3. Launch server: `yarn dev`
